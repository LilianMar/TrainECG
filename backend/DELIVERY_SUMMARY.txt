## 🚀 Cómo Usar

### Opción 1: Setup Automático (Recomendado)
```bash
cd ~/Desktop/Tesis/BACK/ecg-backend
bash setup.sh
python run.py
```

### Opción 2: Manual
```bash
cd ~/Desktop/Tesis/BACK/ecg-backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python run.py
```

### Opción 3: Con Docker
```bash
cd ~/Desktop/Tesis/BACK/ecg-backend
docker-compose up -d
```
